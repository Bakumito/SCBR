Comandos frequentes

  • pwd (Mostra o caminho atual do diretório)
  • ls (mostra o conteúdo do diretório)
  • git init (inicia o tracking do git)
  • touch exemplo.html (cria um arquivo)
  • start code exemplo.html (cria um arquivo na pasta)
  • git add (adiciona o arquivo ao stage de trabalho.)
  • git commit -m "mensagem" (upa para a branch master)
  • git status (verifica o status atual dos arquivos)
  • git diff (mostra as alterações no) arquivo
  • git log (mostra um log dos commit)
  • git rm --cached nomedoarquivo.txt (remove um arquivo do stage sem deletar o arquivo)
  • rm nomedoarquivo.txt -f (deleta um arquivo)
  • rm -rf nomedapasta/ (para excluir a pasta e todo os seu conteudo.)
  • git revert HasHdOcoMmiT (reverte um commit)
