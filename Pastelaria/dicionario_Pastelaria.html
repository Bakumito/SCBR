As entidades supracitadas se relacionam da seguinte maneira:

Entidade Relacionamento Relacionado
Usuario Reside Endereco_Usuario
Usuario Pertence Telefone_Usuario
Usuario Possui Cargo
Usuario Cadastra Usuario
Endereco_Usuario Define Tipo_Logradouro
Telefone_Usuario Depende Tipo_Telefone
Cargo Possui Area_Departamento
Cargo Compoe Cargo_Tarefa
Cargo Cria Tarefa
Cargo_Tarefa Realiza Tarefa

Atributos

Tabela Usuario

Atributo Tipo de Dados Comprim. Restrições Descrição
ID_Usuario Inteiro 4 bytes PK, NOT NULL Identificação do usuario na empresa.
Email Caractere 80 bytes NOT NULL Email do usuario, usado como login
Senha Caractere 12 bytes NOT NULL Senha de login
Nome_Usuario Caractere 20 bytes NOT NULL Nome do usuario
Sobrenome_Usuario Caractere 40 bytes NOT NULL Sobrenome do usuário
Nascimento Data 8 bytes NOT NULL Data de nascimento do usuário
Foto Caractere 200 bytes NOT NULL Endereço da hospedagem da foto do usuario
CPF Caractere 11 bytes NOT NULL, UNIQUE CPF do usuário
Status_Usuário Caractere 20 bytes NOT NULL Status do usuário (estagio, férias, ativo, afastado, demitido)

Tabela Endereco_Usuario

Atributo Tipo de Dados Comprim. Restrições Descrição
Cod_End_Usuario Inteiro 4 bytes PK, NOT NULL Codigo do endereço do usuário, gerado automaticamente
CEP Caractere 8 bytes NOT NULL CEP do endereço do usuário
Cod_Tipo_Logradouro Inteiro 4 bytes FK, NOT NULL Codigo do tipo do logradouro do usuário, usado como chave estrangeira para a tabela Tipo_Logradouro
Nome_Rua Caractere 40 bytes NOT NULL Nome da rua do usuário
Num_Rua Inteiro 4 bytes NOT NULL Número do edifício do usuário
Complemento caractere 40 bytes NULL Complemento do endereço do usuário
ID_Usuario Inteiro 4 bytes FK, NOT NULL Identificação do usuario na empresa, usado como chave estrangeira para a tabela Usuario.

Tipo_Logradouro

Atributo Tipo de Dados Comprim. Restrições Descrição
Cod_Tipo_Logradouro Inteiro 4 bytes PK, NOT NULL Codigo do tipo do logradouro, gerado automaticamente
Tipo_Logradouro Caractere 8 bytes NOT NULL Tipo do logradouro (Rua, Av., Praça, etc)

Telefone_Usuario

Atributo Tipo de Dados Comprim. Restrições Descrição
Cod_Telefone Inteiro 4 bytes PK, NOT NULL Codigo do telefone do usuário, gerado automaticamente
Cod_Tipo_Telefone Inteiro 4 bytes FK, NOT NULL Codigo do tipo de telefone, chave estrangeirapara a tabela Tipo_Telefone
Num_Telefone caractere 11 bytes NOT NULL Número do telefone do usuário
ID_Usuario Inteiro 4 bytes FK, NOT NULL Identificação do usuario na empresa, usado como chave estrangeira para a tabela Usuario.

Tipo_Telefone

Atributo Tipo de Dados Comprim. Restrições Descrição
Cod_Tipo_Telefone Inteiro 4 bytes PK, NOT NULL Codigo do tipo de telefone, chave primária gerado automaticamente
Tipo_Telefone Caractere 20 bytes NOT NULL Tipo do telefone do usuário (residencial, celular, comercial)

Area_Departamento

Atributo Tipo de Dados Comprim. Restrições Descrição
Cod_Departamento Inteiro 4 bytes PK, NOT NULL Codigo do departamento, chave primária gerado automaticamente
Nome_Departamento Caractere 40 bytes NOT NULL Nome do departamento
Cod_Cargo Inteiro 4 bytes FK, NOT NULL Codigo do cargo, usado como chave estrangeira para a tabela Cargo

Cargo

Atributo Tipo de Dados Comprim. Restrições Descrição
Cod_Cargo Inteiro 4 bytes PK, NOT NULL Codigo do cargo, chave primária gerado automaticamente
Nome_Cargo Caractere 40 bytes NOT NULL Nome do cargo (Estagiario, gestor junior, gestor senior, gerente de produtos, etc)
ID_Usuario Inteiro 4 bytes FK, NOT NULL Identificação do usuario na empresa, usado como chave estrangeira para a tabela Usuario.

Cargo_Tarefa

Atributo Tipo de Dados Comprim. Restrições Descrição
Cod_Cargo Inteiro 4 bytes PK, FK, NOT NULL Codigo do cargo, chave primária e estrangeira, fazendo ponte entre as tabelas Cargo e Tarefa
ID_Tarefa Inteiro 4 bytes PK, FK, NOT NULL ID da tarefa, chave primária e estrangeira, fazendo ponte entre as tabelas Cargo e Tarefa

Tarefa

Atributo Tipo de Dados Comprim. Restrições Descrição
ID_Tarefa Inteiro 4 bytes PK, NOT NULL ID da tarefa, chave primária gerada automaticamente
Cod_Cargo Inteiro 4 bytes FK, NOT NULL Codigo do cargo, chave estrangeira para a tabela Cargo
Mensagem Caractere 200 bytes NOT NULL Descrição da tarefa
Data_Inicio data 10 bytes NOT NULL Data de criação da tarefa
Data_Limite data 10 bytes NOT NULL Data limite para a finalização da tarefa
Data_Fim data 10 bytes NULL Data em que a tarefa foi finalizada
Status_Tarefa Caractere 10 bytes NOT NULL Status atual da tarefa (Não inicializado, Em andamento, Finalizado)
